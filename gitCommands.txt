      To Push to you Git repo

1. Initilize Git Repo :                           git init
2. Add all file in the local repo to git index:   git add .

(if you are doing first time:)
   git config --global user.email "email_here"
   git config --global user.name "Name_here"

3. commit the changes to the local repo:          git commit -m "initial commit"
4. Add the remote repo to the local repo:         git remote add origin <repo_URL>
5. push the changes to the git repo     :         git push -u origin master

      To Pull on you Directory from Git
   
1. git fetch origin
2. git pull origin <branch_name>
